# CSharp.EchoHubSignalR

Démonstration de communication temps réel avec SignalR : un hub d'écho qui renvoie les messages reçus à tous les clients connectés.

## Structure

- `EchoHubSignalR/Echo.cs` — Hub SignalR (serveur)
- `EchoHubSignalR/Index.htm` — Client web avec jQuery
- `EchoHubSignalR/Scripts/` — jQuery 1.7.2 + SignalR 0.5.0

## Stack

C# / ASP.NET / SignalR / jQuery
